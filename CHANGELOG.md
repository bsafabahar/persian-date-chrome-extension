# تغییرات (Changelog)

- الگوریتم تبدیل تاریخ شمسی دقیق‌تر شده است
- عملکرد بهتر در شناسایی و تبدیل تاریخ‌ها

### 🧹 پاک‌سازی

- کد تمیز و آماده تولید

### 🐛 رفع مشکلات
- رفع مشکل تبدیل نادرست برخی تاریخ‌ها
- بهبود پایداری و کارایی
- حذف خطاهای احتمالی JavaScript
